YAZILIM DİLLERİ

HTML web sitelerinin temelini oluşturmuştur. HTML5 ise en son sürümüdür.Temel görevi sitemize yazı, görüntü, video gibi değişik verileri bağlamaktır. Bununla beraber diğer tüm tarayıcılara standart ve düzgün olarak görüntülemesi için kurallar belirler. HTML kodlarını web tarayıcıları okur, yorumlar ve ziyaretçilerin görsel anlamda inceleyebileceği hale dönüştürürler. Her tarayıcı bu yüzden kendine has farklı sonuçlar verebilir. Günümüzde web tasarım personellerinden işe alım ilanlarında istenen en temel gereksinim çalışmalarını tüm tarayıcılara uyumlu olacak şekilde geliştirmeleri istenir.

HTML 5 NEDİR ? NEDEN ÖĞRENMELİYİZ

CSS, (Cascading Style Sheets) yani, basamaklı stil şablonları anlamına gelmektedir. CSS bir biçimlendirme dilidir. Web site sayfalarını kendi stil kurallarına göre şekillendirebilme olanağı verir. Kendine özgü kuralları bulunan CSS, javascript ve HTML ile üst düzey web sayfaları hazırlanabilir. Web tasarım alanında yoğun olarak kullanılan CSS, html kavramından sonra karşınıza çıkan ilk biçimlendirme dilidir. HTML (HyperText Markup Language) ile birlikte entegre şekilde çalışan CSS, oluşturduğunuz web sayfalarında yapacağınız özellikleri geliştirmenize olanak sağlar. Web sayfalarının renklerini, fontlarını, görsel efektlerini, mizanpajını ve tüm öğeleri CSS stil şablonlarıyla biçimlendirebilirsiniz

CSS NEDİR ? NASIL YAZILIR ?

Açılan veya kayan menüler, hareketli semboller vs. web sayfasını dinamik bir hale getirir. Javascript ile oluşturulan web sayfaları, estetik olarak daha güzel görünür ve kullanıcılar tarafından beğeni toplar.
